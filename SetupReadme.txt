## Setup

1. **Prerequisites**:
   - Ensure you have **Node.js** installed on your machine. You can download it from Node.js official website.

2. **Extract the Project**:
   - Download and extract the provided ZIP file containing the project.

3. **Install Dependencies**:
   - Navigate to the project directory(backend and jobsheeker,one by one) and run the following command to install all necessary dependencies:

     npm install
    

4. **Technologies Used**:
   - **Backend**: Express.js
   - **Database**: MongoDB
   - **Frontend**: React
